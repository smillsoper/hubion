06 — Usage Rules

Do's & Don'ts

✓ Do

  • Use the gradient (Sky 500 → Sky 700) for primary CTAs
  • Maintain clear space around the logo equal to the height of the "C"
  • Use Inter for all UI and marketing text
  • Use JetBrains Mono for all technical values, IDs, and code
  • Use the dark background version on hero sections
  • Keep "Contact" bold and "Connection" light in the wordmark always
  • Use semantic colors consistently (green = active, red = error)
  • Use the nav bar logo variants at heights of 48–56px

✕ Don't

  • Stretch, skew, or rotate the logo or icon
  • Place the logo on busy photographic backgrounds
  • Use brand blue as a background color for large areas
  • Change the font weight split in the wordmark
  • Add a space between "Contact" and "Connection" in the wordmark
  • Use more than two type sizes in a single card or component
  • Use non-brand colors for interactive elements
  • Apply drop shadows to the wordmark
